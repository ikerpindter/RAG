# RAG híbrido con evals y citations sobre reportes financieros

Responde preguntas sobre reportes 10-K combinando retrieval híbrido con generación que cita sus fuentes, y mide la calidad de las respuestas con un set de evals.

## Stack

- Python 3.12
- [uv](https://docs.astral.sh/uv/) como gestor de entorno y dependencias
